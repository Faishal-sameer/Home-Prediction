# Home-Prediction
Predict house prices based on different features like area, bedrooms, location, etc.

🚀 **My First Python Regression Project!**

I’m excited to share that I’ve completed my **first Linear Regression project using Python** 🎉

📊 **Project:** House Price Prediction
📌 Goal: Predict house prices based on different features like area, bedrooms, location, etc.

---

🔍 **What I did:**

* Cleaned and prepared the dataset
* Performed Exploratory Data Analysis (EDA)
* Built multiple models:

  * Linear Regression
  * Lasso Regression
  * Ridge Regression
* Evaluated models using R² Score and RMSE

---

📈 **Key Results:**

* Achieved an R² score of **0.98**
* Linear Regression performed best with strong accuracy and simplicity

---

💡 **Key Learnings:**

* Importance of feature relationships in prediction
* How regularization (Lasso & Ridge) works
* Model evaluation and comparison
* Translating data into meaningful insights

---

📊 This project helped me understand how machine learning can solve real-world business problems like pricing and forecasting.
